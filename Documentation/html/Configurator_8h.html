Indicates a position ignoring spaces.

Gives the value of the line (specific line)

From a line like "Key : value", the function returns the value.
Note that it just returns the string after spaces and one separator.
A separator (':' for example) could be given.

Parameters

separator: An optional separator between key and value can exists. Give ' ' if you haven't one.
