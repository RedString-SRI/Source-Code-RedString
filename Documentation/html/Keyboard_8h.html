int getKeyboard_Double ( double *  typedDouble,
double  min,
double  max 
)

Allow the user to type a double.

It allows the user to type typedDouble. This function is secured, the double must be within min and max. Otherwise, the function returns an error.
The function returns errors in those following cases :

  • A non digit (except the space not taken into account) is typed
  • The user types too much digits
  • The user does not type a digit but press 'Enter'
  • The typed double is outside min or max
  • An unknown error has occurred
    Parameters
    typedDoubleVariable for the typed double. 0 is returned if an error has occurred.
    minThe minimum you want for typedDouble (not inferior to HUGE_VALL)
    maxThe maximum you want for typedDouble (not superior to HUGE_VALF)
    Returns
    1 : Success
    0 : An unknown error has occurred (or an error from getKeyboard_String)
    -1 : The user types a double superior to max
    -2 : The user types a double inferior to min
    -3 : The user does not type a digit but press 'Enter'
    -4 : A non-digit (except the space) is typed

int getKeyboard_String ( char *  typedString,
int  minChar,
int  maxChar 
)

Allow the user to type a string.

It allows the user to type typedString. This function is secured, the string must contains within minChar and maxChar characters (this size must be inferior to (strlen(typedString)).
When allocating typedString, you must take in consideration an ending character '
' needed by fgets but the typedString won't be returned with it. Otherwise the function returns an error.
The function returns errors in those following cases :

  • The user types too much characters
  • The user types not enough characters
  • An unknown error has occurred
    Parameters
    typedStringVariable for the typed double
    minCharThe minimum you want for typedString (not inferior to 0 and not superior to maxChar)
    maxCharThe maximum you want for typedString (not inferior or equal to minChar)
    Returns
    1 : Success
    0 : An unknown error has occurred
    -1 : The user types too much characters
    -2 : The user types not enough characters
